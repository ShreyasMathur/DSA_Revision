# DSA Revision in C++

This project is a hands-on implementation and analysis of fundamental Data Structures and Algorithms in C++.

It focuses on:
- Implementing sorting and searching algorithms
- Measuring time and operation counts
- Building intuition for time complexity through experimentation

## Algorithms Implemented

### Sorting
- Merge Sort
- Quick Sort
- Selection Sort
- Insertion Sort
- Bubble Sort

### Searching
- Binary Search (Recursive)

## Time Complexity Comparison

| Algorithm       | Best Case | Average Case | Worst Case | Stable |
|----------------|----------|-------------|------------|--------|
| Merge Sort     | O(n log n) | O(n log n) | O(n log n) | Yes |
| Quick Sort     | O(n log n) | O(n log n) | O(n²)      | No |
| Insertion Sort | O(n)       | O(n²)      | O(n²)      | Yes |
| Selection Sort | O(n²)      | O(n²)      | O(n²)      | No |
| Bubble Sort    | O(n²)      | O(n²)      | O(n²)      | Yes |
